Welcome to the NDEx BETA Test Site

This is the test site for NDEx, the Network Data Exchange.

NDEx is open-source, web-based software system to enable individuals, groups, and software applications to share, store, manipulate, and publish biological network knowledge.

This test site will allow users to create accounts and upload networks, the site and its underlying database are not guaranteed to be stable or persistent. If you create an account and upload content, that content may not be present when you return to the site - during the development process, we will frequently need to re-create and reload the test database.

If you encounter problems while testing, you can report them to NDEx Support. We would especially appreciate an email if you encounter any situation in which the site becomes inoperable.

About NDEx

NDEx is a project of the Cytoscape Consortium in conjunction with the Ideker Lab at the UCSD School of Medicine.

Learn more about NDEx at the NDEx Project Site.

Techincal details for NDEx are documented at the NDEx Github Repository Wiki.
